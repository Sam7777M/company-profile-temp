# Frontend Task - Professional Services Website

A modern, responsive React application built from a Figma design with a dark brown/earthy color scheme.

## Features

- **Header Navigation**: Sticky header with navigation links (Home, About, Services, Portfolio, Blog, Contact)
- **Hero Section**: Eye-catching hero section with gradient background and call-to-action
- **Service Details**: Grid layout showcasing services with hover effects
- **Profile Section**: Team member profiles with images and descriptions
- **Frame 3**: Two-column layout with Tool Introduction and Key Features
- **Footer**: Comprehensive footer with links and company information
- **Responsive Design**: Fully responsive across all device sizes
- **Dark Theme**: Professional dark brown/earthy color palette

## Tech Stack

- React 18
- Vite
- CSS3 (Custom CSS with CSS Variables)
- Modern ES6+ JavaScript

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

The production build will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```
